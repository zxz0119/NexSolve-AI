## ⚡ 项目概述

**NexSolve AI** 是一个建立在 GitHub 上的去中心化“需求广场”。

我们致力于连接两端：
- **需求方**：传统行业（餐饮、零售、物流等）的小老板，有痛点但不懂技术。
- **开发者**：擅长利用 AI 工具（Cursor、Claude、GPT）快速交付方案的极客。

---

## 🛠 工作流程

1. **提交痛点**：用户点击 Issue，使用“痛点需求”模板提交，或通过 [网页端](https://xk7yf4c2cf.coze.site/) 快速发布。
2. **开发者认领**：开发者回复“正在处理”并打上 `in-progress` 标签。
3. **交付成果**：开发者在回复中提交代码、演示视频或工具链接。
4. **协商报酬**：双方私下通过微信/支付宝等方式结算，平台不参与抽成。
5. **案例沉淀**：优秀的方案将进入“案例库”展示。

### 🏷️ 标签指南

为了提高协作效率，我们使用以下标签来标识需求状态：

- ![pain-point](https://img.shields.io/badge/pain--point-d73a4a) : **待解决的原始需求**（系统自动挂载）。
- ![needs-triage](https://img.shields.io/badge/needs--triage-e99695) : **待处理**，等待管理员进行初步核实。
- ![in-progress](https://img.shields.io/badge/in--progress-1D76DB) : **开发中**，已有开发者认领。
- ![solved](https://img.shields.io/badge/solved-008672) : **已解决**，方案已交付。
- ![bounty-open](https://img.shields.io/badge/bounty--open-D4C5F9) : **带赏金**，建议开发者优先关注。
