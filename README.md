## ⚡ 项目概述

**NexSolve AI** 是一个建立在 GitHub 上的去中心化“需求广场”。

我们致力于连接两端：
- **需求方**：传统行业（餐饮、零售、物流等）的小老板，有痛点但不懂技术。
- **开发者**：擅长利用 AI 工具（Cursor、Claude、GPT）快速交付方案的极客。

---

## 🛠 工作流程

1.  **提交痛点**：用户点击 Issue，使用“痛点需求”模板提交。
2.  **开发者认领**：开发者回复“正在处理”并打上 `in-progress` 标签。
3.  **交付成果**：开发者在回复中提交代码、演示视频或工具链接。
4.  **协商报酬**：双方私下通过微信/支付宝等方式结算，平台不参与抽成。
5.  **案例沉淀**：优秀的方案将进入“案例库”展示。
